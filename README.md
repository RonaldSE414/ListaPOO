# ListaPOO

Lista de Exercícios – Programação Orientada a Objetos (Java)

Este repositório contém a resolução de 10 questões práticas da disciplina de Programação Orientada a Objetos.

Os exercícios abordam conceitos fundamentais como:
	
•	Criação de classes e objetos
	
•	Construtores
	
•	Métodos
	
•	Regras de negócio
	
•	Interação entre múltiplas classes

Cada questão foi implementada em um arquivo separado, utilizando o nome da classe pública como nome do arquivo.
